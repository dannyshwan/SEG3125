# SEG3125 Module 6 Lab

## How to run
1. Clone repository
2. Install packages
   ```bash
      npm install
   ```
3. Start by using the following command in a terminal
   ```bash
      npm run start
   ```

   OR
   
   ```bash
      node app.js
   ```

4. To view the survey, go to: http://localhost:3000/nicesurvey
5. To view the results/data, go to: http://localhost:3000/analysis
